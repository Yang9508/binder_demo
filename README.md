# demo

1. matrix.py
- calculate column mean of a matrix:
```
[1 2 3]    [   3  ]
[4 5 6] to [4.6667]
[6 9 8]    [   5  ]
```

- save the mean as `demo_matrix.csv`.
2. plot.R
- read `demo_matrix.csv` 
- add a list `year = c(1990,2000,2010)`
- plot data with x=year, y=mean
- save the plot as `demo_plotr.png`.
